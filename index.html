How to Play the Dice Game

  • Use the Roll dice button to start, Player 1 starts the initial game until there's a winner.
  • Use the Hold button to save the score for the active player, this will save and switch to the next player.
  • Use the New Game button to start another game after a win. Once there is a winner, it start a fresh game and switch the winner as active player.
  • Dice 1 (one) switches the player and cancels other players current score not the winning tracks. The winning track is saved on the players top left or right corner.
